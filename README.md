# Ebay_clone backend

## Features:
 * Nodemon - server auto-restarts when code changes
 * Koa 
 * TypeORM 

### Getting Started
- Clone the repository
```
git clone https://github.com/vdagraca/ebay_clone.git
```
- Install dependencies
```
cd <project_name>
npm install
```
- Run the project
```
nodemon .
```

Backend is deployed to heroku: https://ebayclone.herokuapp.com/
